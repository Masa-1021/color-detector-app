## キーワード集

| 用語 | 読み方 | 意味 |
|------|--------|------|
| MQTT | エムキューティーティー | 軽量なメッセージングプロトコル |
| Broker | ブローカー | メッセージを仲介するサーバー |
| Publisher | パブリッシャー | メッセージを送信する側 |
| Subscriber | サブスクライバー | メッセージを受信する側 |
| Topic | トピック | メッセージの分類（住所） |
| Pub/Sub | パブサブ | 発行/購読モデル |
| Mosquitto | モスキート | 人気のMQTTブローカー |
| QoS | キューオーエス | サービス品質レベル |
| Payload | ペイロード | メッセージの中身（データ） |
